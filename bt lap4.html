WritePad Editor

WritePad is an edior application that allows you to create Wed pageseasily and quickly.

Steps to open WritePad application:

  1. Click Start
  2. Click Programs
  3. Click WritePad
Advantages of using a WritePad editor:

  • It allows faster creation of Wed pages.
  • It allows easier updation of text or images in WritePad editor.
  • It is platform independent.
Disadvantades of using a WritePad editor:
  • It consumes more memory spase.
  • It requires to fill-in more details such as meta data for each page.
